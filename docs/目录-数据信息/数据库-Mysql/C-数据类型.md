### 数据类型

概览

```
三类：数值、日期|时间、字符串
```

#### 数值

```
TINYINT
	取值范围：-128~127
	占用空间：1字节
	
SMALLINT
	取值范围：-32768，32767
	占用空间：2字节
	
MEDIUMINT
	取值范围：-8388608，8388607
	占用空间：3字节
INT
	取值范围：-2147483 648，2147483 647
	占用空间：4字节
BIGINT
	取值范围：-9223372036854775808，9223372036854775807
	占用空间：8字节
	
FLOAT
	取值范围：3.402823466 E+38,1.175 494 351 E-38
	占用空间：4字节
DOUBLE
	取值范围：-1.7976931348623157 E+308,1.175494351 E-38
	占用空间：8字节
DECIMAL(M,D)
	
```

#### 字符串

```
参数M:最多存储M个字符，按个数算
占用内存空间:
	按字符大小算
	1个英文占用 1byte，一个汉字占用 2bytes
	char最多存储255个英文字符，127个中文字符
	
CHAR(M)
	内存空间:0-255 字节
	特点:定长字符串，效率高
		不足M个字符填充空格
		取数据时，用trim去除空格
VARCHAR
	内存空间:0-65535 bytes
	特点:可变长度
tinytext
	内存空间:0-255 字节
```
