## 汇编-基础

### 概览

```
·	汇编组成:汇编指令，伪指令，其他符号
·	总线:
		分类:地址总线，数据总线，控制总线;
		作用:用于区分指令
·	每种CPU都有自己的汇编指令集
·	存储器都是从0开始的
·	寻址能力:最多可以表达多少种数据
·	64位CPU有16个寄存器
```

### 内存地址分布

经典8086CPU

| 名称         | 地址        | 占用空间 |
| ------------ | ----------- | -------- |
| 主随机存储器 | 0~7FFFH     | 32KB     |
| 显存         | 8000H~9FFFH | 8KB      |
| 其他ROM      | A000H~FFFFH | 24KB     |
