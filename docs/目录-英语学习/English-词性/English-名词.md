## 专有名词

#### **特性**

```
	1.只有两个以上的普通名词组成的专有名词前，一般要用定冠词。
	ex:the Great Wall(长城)。
	
	2.姓氏名如果采用复数的形式，则表示该姓氏一家人(复数含义)。
	ex：the Greens(格林一家人)
	
```

## 普通名词

#### **个体名词**

```
car(汽车)、room(房间)、 fan(风扇)、photo(照片)
```

###### 单复数变化

```
(1)一般情况下，在名词的词尾直接加s.

(2)以s,x,ch,sh结尾的名词变复数时，要在词尾加es.

(3)以“辅音字母+y”结尾的名词，先变y为i,再加es.

(4)以单个O结尾的可数名词单数变复数基本遵循词义若是有生命个体则加es,若词义是无生命个体则加s(如photns,radios,pianos等),同时，须十分注意，若是以双个O结尾，不管其词义是否为有生命个体，都只能加s(如kangaroos,zoos等).

(5)以f或fe结尾的名词变复数时，先把f或fe变为v,再加es.
```



#### 集体名词

**表示类的集体名词**

```
语法：
	1.形式上总是单数，没有复数形式，作主语时，谓语用单数。
	The old machinery is out of date.
	
	2。不可直接与a（n）或数词连用。表数量时用much，little，some，a piece of，an article of,a set of等。
	Each room has five pieces of furniture.
	
	3.若需用代词，用单数代词
	Do you want to see my jewelry?It is in the box.
	
	4.有单复数的变化
单词
clothing，furniture，baggage/luggage，jewelry
traffic，information，machinery，merchandise
produce，scenery，vegetable，fruit，ice cream
```





**表示"群"的集体名词**

```
语法：
	1.上述(Ⅰ)部分的词作主语时，谓语动词通常用复数。
	The police are looking for him
	
	2.上述(Ⅱ)部分的词把集体当作一个整体时，用作单数；若考虑构成这些集体的各个成员时，看作复数。
	My family is a large one.
	
	3.把一个集体名词看作单数或复数，要注意前后一致
	The team is famous for its(不能用their) long history.

(Ⅰ)
people,police,cattle,poultry (家禽）,vermin （害虫，寄生虫）
clergy （牧师，神职人员）,militia（民兵组织）,crew 队，全体人员，全体成员
(Ⅱ)
family,class,team,government,crowd,committee
jury,party,firm,couple,board,group,gang,enemy,union
audience,public,mankind,humanity,youth
```

#### 物质名词

```
语法:
	1.一般没有复数形式
	2.只有特指时加the
	3.借助量词表达数量
	
①气体：air，gas，smoke，steam

②液体：water, oil, tea, coffee, wine

③粉状或细粒状物品：flour，salt，sugar，rice，sand

④块状物体（品）：常见的块状物体（品）可再区分如下：
⑴食品：bread，butter，chocolate，meat,chicken，mutton，pork

⑵日常用品：chalk，paper，money，soap

⑶矿物：gold，iron，silver

⑷原料/材料：brick，clay[clei]，cloth，cotton，earth，glass，stone，wood，wool
```
