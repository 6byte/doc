## 常量池

概览

```
种类:
	全局字符串池,class文件常量池,运行时常量池
```

### 全局字符串池

```
时机:经过类加载，验证，准备之后
内容:引号中的字符串，final修饰的字符串
特点：
    全局字符串池	是一个StringTable类，它是哈希表
    全局字符串池	实列仅有一份，且能被全局共享
    全局字符串池	存放引用值，堆存放具体值
```

### class文件常量池

```
时机:在编译成class文件之后，生成class常量池
内容:
	--	符号引用
        类和接口的全限定名
        字段的名称和描述符
        方法的名称和描述符
```

### 运行时常量池

```
时机:类执行阶段将class常量池存放运行时常量池
```
